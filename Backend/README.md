# HITCON 2026 NFC Tag Game Backend API

The backend API specification is maintained in [`openapi.yaml`](./openapi.yaml).

Use Swagger UI to view and explore the API contract:

1. Open <https://editor.swagger.io/>.
2. Import or paste the contents of [`openapi.yaml`](./openapi.yaml).
3. Use the rendered Swagger UI panels to inspect endpoints, schemas, examples, authentication, and error responses.

`openapi.yaml` is the canonical source of truth for backend API documentation.
